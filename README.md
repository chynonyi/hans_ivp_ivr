#### Ручной запуск бэктеста
Можно запустить тест отдельно как ты и просил, для конкретной акции с нужными параметрами, игнорируя оптимизацию.
1. Открой `backtest.py`.
2. В самом верху раскомментируй и заполни блок `MANUAL_SETTINGS`:
```python
MANUAL_SETTINGS = {
    "Symbol": "TSLA",
    "IVP_from": 30,
    "IVP_to": 60,
    "Type": "Stock"
}
```
3. Запусти скрипт и он выполнит тест только для этой настройки.
